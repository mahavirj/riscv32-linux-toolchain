# riscv32-linux-toolchain

Prebuilt **RISC-V 32-bit Linux** cross-toolchain (GCC + musl) for
`rv32imac` / `ilp32` (soft-float) targets.

A generic `riscv32-buildroot-linux-musl` toolchain — there is nothing
chip-specific in it. Published so builds can download a ready toolchain
instead of compiling one from source.

Toolchains are attached to the [Releases](../../releases) page.
